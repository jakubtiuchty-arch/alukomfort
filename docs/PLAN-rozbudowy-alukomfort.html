1. Diagnoza

1.1. Stan obecny strony

  • Tech stack: statyczna React + Babel via CDN (bez build), 1 strona index.html, JSX serwowany przez prosty HTTP server.
  • Repo: github.com/jakubtiuchty-arch/alukomfort
  • Routing: hash-based (#/produkty/linea...), brak SSR.
  • Strony aktywne: / (Home), /produkty/linea, /produkty/horizon, /produkty/roma.
  • Czego nie ma: /o-nas, blog, FAQ, realizacje (galeria), strony lokalne, cennik, llms.txt, schema markup, video YouTube, opinie, konfigurator.
  • Producent: PLAST-MET Sp. z o.o. Sp. k., ul. Milicka 34, 55-100 Trzebnica (40 km od Wrocławia).
  • Certyfikat: TÜV NORD nr 2274-CPR-0046-2025 (PN-EN 1090-1, klasa EXC 2).
